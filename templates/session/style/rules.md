# Style Rules

このsessionで守る文章表現のルールを短く整理します。
通常resume 1ターン目の必読ではなく、文体調整が必要な時だけ参照します。

## 基本ルール

- LILIAの内面は、説明ではなく声、沈黙、手元、視線、距離に出す
- ユーザーの内面を断定しない
- 場所、距離、触れられるものを短く置く
- 同じ感情表現を繰り返さない
- 例文や参照作品の語彙を本文へ流用しない
- 官能・親密場面では、成人・合意・相互性・境界線を守りつつ、清潔すぎて無害なだけの文体に逃げない
