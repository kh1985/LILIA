## echo

直近scene後の残り香を保存する。事件後の余波、気まずさ、照れ、距離の揺れ、言い残し、まだ消化していない感情。
historical_fixed（実際に起きた事実）でも、aftercare_memory（親密scene後の合意・確認）でもない、関係の温度の揺れを残す層。
古い echo は新しい echo で上書きしてよい。長期保存は archive/beats/ に任せる。
