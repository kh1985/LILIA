# Opening Pattern Stock

著名な小説・RPG・ノベルゲー・映画の冒頭から抽出した**開幕の構造パターン**。
作品単位ではなく、ジャンルをまたいで観察される「始め方の形」を集めたもの。

## このファイルと既存 reference との違い

| ファイル | 役割 | たとえると |
|---------|------|----------|
| `story_pattern_stock.md` | 物語全体の形（12パターン） | コース全体の流れ |
| `story_structure_stock.md` | 普遍的構造論（5構造） | 料理の型 |
| `opening_pattern_stock.md` | 冒頭シーン専用の形（15パターン） | 1皿目の出し方 |

物語全体の構造と冒頭の構造は別物。「全体は P1 距離の縮小」でも「冒頭は O3 音で開く」でよい。

## 利用ルール

- 1セッションに1〜2パターンまで（A群 / B群 / C群 / D群 から各最大1つ）。混ぜると焦点が散る。
- パターン名や作品名を本文に出さない（GM の内部ノート）。
- パターンは「冒頭で何を取りに行くか」を決めるため。「何でも書ける」状態を避ける道具。
- Q&A の素材（世界、ヒロイン、出会い）に合わないパターンを無理に当てない。
- 例は構造説明のみ。literal として真似しないこと。
- 冒頭は 8〜15 文の範囲を超えない。1パターン分の負荷で書ききる。

## 冒頭が果たすべき4つの仕事

すべての冒頭はこの4つを最低 2 つは満たす:

1. **Hook（掴み）**: 読み手が次の文を読みたくなる引っかかり
2. **Orientation（方位）**: 主人公が今どこにいて、何者かの最低限の手がかり
3. **Agency Placement（プレイヤーの立ち位置）**: プレイヤーが「観察者」「介入者」「当事者」のどれかを決める
4. **Unresolved Hook（未回収の鈎）**: 次のターンへ持ち越す問い

冒頭でこれらを「説明」してはいけない。**配置する**。

## A群: 入口の取り方（最初に何を見せるか）

### O1. 場所で開く

**形**: 風景・空間の一文で物語の場を一発で開く。主人公の名前や状況より先に、**世界の温度**を確定させる。読み手は「自分がいま、どこにいるか」を場の感触から得る。

**観察される作品**: 川端康成『雪国』, 村上春樹『ノルウェイの森』, トールキン『指輪物語』冒頭, AIR(VN) 旅人視点の浜辺, FF10 ザナルカンド遠景

**例（構造説明のみ。literal として真似しないこと）**:
- [雪/雨/夕焼け] の温度の一文 → 主人公が「ここ」にいることが示される → 動きが始まる
- 通り抜けの境界（トンネル、橋、坂、扉）を超える描写 → 場が切り替わる → 異界感
- 遠景で都市・自然を映す → カメラが主人公の手元へ寄る → 個別の動作が始まる

**使い時**: 世界の質感を Q&A から拾えている時。場が物語の主役に近い時（季節、街、施設が重要）。

**注意**: 描写が綺麗でもプレイヤーが「自分は誰か」を取れないと迷子になる。3 文以内で人称を立てる。

---

### O2. 物に焦点

**形**: 小さな物から物語が立ち上がる。物が**世界の縮図**として置かれ、それをめぐる視線・手の動きが冒頭の動きになる。物は後で意味が回収される。

**観察される作品**: 多くのミステリ冒頭（凶器・遺品・手紙）, 川端『山の音』, 安部公房『砂の女』の道具, ToTW（Trails of the Walking）系の遺物, ペルソナ5の手錠

**例（構造説明のみ。literal として真似しないこと）**:
- [小道具] が机に置いてある描写 → 持ち主が問われる → 持ち主は冒頭では出ない
- [日用品] の使い方が普通と少し違う → 違和感の種 → 後の reveal の入口
- [古い物] の傷の描写 → どう傷ついたかは語らない → ghost の予告

**使い時**: Q3「描写の縛り」で物的アンカーが取れている時。その物自体を冒頭の主役に格上げする。

**注意**: 物が3個以上出ると焦点が散る。1個に絞る。

---

### O3. 音・感覚で開く

**形**: 視覚以外の感覚（音、匂い、温度、皮膚感）を最初に置く。視覚情報は後から追いつく。読み手は**意味より先に状態**を受け取る。不穏・違和・心地よさを身体で先取りさせる。

**観察される作品**: ひぐらしのなく頃に（蝉時雨）, Steins;Gate（街のノイズ）, 村上龍『限りなく透明に近いブルー』感覚描写, 多くのホラー冒頭

**例（構造説明のみ。literal として真似しないこと）**:
- [音/匂い] が先 → どこから来るかは保留 → 視覚が遅れて追いつく
- [温度/湿度] が肌で先に来る → 場所の名前は後 → 季節と気分を確定
- [遠くの音] と [近くの音] のレイヤー → 空間の奥行き → 次に何が来るかの予感

**使い時**: 不穏や色気を冒頭に置きたい時。説明文に逃げず、身体で確定させたい時。

**注意**: 感覚を3つ以上重ねると詩になりすぎる。2感覚までで止める。

---

### O4. 起床から始める

**形**: 主人公が眠りから醒める瞬間を冒頭にする。意識の立ち上がりと、世界が再起動する瞬間を重ねる。プレイヤーは主人公と同じ速度で世界を取り戻す。

**観察される作品**: クロノ・トリガー（母に起こされる）, MOTHER2（流れ星で目覚める）, ペルソナ4（電車内の半覚醒）, 多くの RPG, 夏目漱石『門』の朝

**例（構造説明のみ。literal として真似しないこと）**:
- [音/光/呼びかけ] で覚醒 → 視界の解像度が上がる → 今日が始まる
- 半覚醒で [異物] を感じる → 完全に起きると消えている → 違和感だけ残る
- [既知のルーチン] の朝 → 1点だけ違うものがある → その1点が物語の入口

**使い時**: 主人公の日常が物語の起点になる時。プレイヤーに「自分が誰か」を最短で取らせたい時。

**注意**: ただ「目が覚めた」だけでは弱い。起きた時に何が違うかを 1 点必ず置く。

## B群: 語り手の置き方（誰の声で始めるか）

### O5. 告白独白

**形**: 一人称の傷・恥・秘密の告白から始める。読み手は冒頭で**共犯**にされる。語り手が信頼できるかは保留される（信頼できない語り手の余地）。

**観察される作品**: 太宰治『人間失格』, サリンジャー『ライ麦畑でつかまえて』, カミュ『異邦人』, 多くの私小説, 月姫（志貴の独白）

**例（構造説明のみ。literal として真似しないこと）**:
- [自分の欠陥] の自己診断 → どう生きてきたかの圧縮 → 今日の状況
- [認めたくない事実] の半告白 → 一部だけ伏せる → 隠した部分が物語の核
- [記憶の断片] の語り直し → 時系列は乱れる → 何かを再構成しようとしている

**使い時**: 主人公自身が物語の主軸になる時。Q5（内面に持っているもの）に重さがある時。

**注意**: 独白が長いとプレイヤーが動けない。3〜5 文で切り、世界に出す。

---

### O6. 名乗りメタ

**形**: 語り手が自己紹介して始める。第四の壁を半分壊して、読み手に直接「これから始まるもの」を提示する。コミカル/誇大/虚構性を冒頭で確定させる。

**観察される作品**: 夏目漱石『吾輩は猫である』, Steins;Gate（岡部の名乗り）, 多くのライトノベル, ファイトクラブの語り

**例（構造説明のみ。literal として真似しないこと）**:
- [大仰な肩書き] の自称 → 実際の状況とのズレ → コミカルな緊張
- [自分が何者か] の宣言 → それを裏付ける/裏切る描写 → ペルソナの提示
- [読み手への呼びかけ] → 物語が「これから始まる」と確定 → 期待を作る

**使い時**: 主人公のキャラクター性が強く、トーンを最初から決めたい時。シリアス過ぎる物語を回避したい時。

**注意**: メタを入れると世界の重みが下がる。シリアスな物語には不向き。

---

### O7. 二人称呼びかけ

**形**: 「あなた」「君」で読み手や主人公を直接指す。冒頭で読み手が**当事者**にされる。一人称の独白とも、三人称の客観とも違う、強い接続を作る。

**観察される作品**: カルヴィーノ『冬の夜ひとりの旅人が』, 一部のCYOA小説, 雫(VN)の一部, 一部の現代詩

**例（構造説明のみ。literal として真似しないこと）**:
- [あなたは今 X している] の宣言 → 状況が確定する → 拒否の余地はない
- [あなたが知らないこと] の提示 → 知りたくなる → ページをめくる動機
- [あなたへの問い] → 答えはまだ出さない → 物語が問いを抱えて進む

**使い時**: プレイヤーの当事者性を冒頭で最大化したい時。実験的なトーンを許容できる時。

**注意**: 多用するとくどい。冒頭の数文だけで切り上げ、通常の語りに戻す。

---

### O8. 同期独白

**形**: 離れた二人（またはそれ以上）のモノローグが交互に重なって、まだ会っていない関係を**先に響かせる**。読み手は二人がいずれ出会うことを冒頭で予感する。

**観察される作品**: 新海誠『君の名は。』, テレンス・マリック『天国の日々』, 一部の青春群像, 並行モンタージュを使う映画全般

**例（構造説明のみ。literal として真似しないこと）**:
- [主人公の独白] / [ヒロインの独白] の交互配置 → 共鳴する語彙 → 距離があるのに通じている感覚
- [異なる場所] の同じ時刻 → 同じ事象（雨、空、音）→ 偶然以上の何か
- [昔の独白] / [今の独白] の重なり → 時間の歪み → 後で意味が回収される

**使い時**: 物語の核に「再会」「同期」「運命」がある時。Q6 が偶然性を含む時。

**注意**: ヒロインの内面を冒頭から書くと「LILIA を完成させない」原則に反する。声色だけ仄めかし、内容は薄く保つ。

---

## C群: 時間の置き方（いつから始めるか）

### O9. 途中から（in medias res）

**形**: 物語の途中（しばしばクライマックス近く）から始める。背景は語らず、**動きの中**にプレイヤーを放り込む。何が起きているかは進行で分かる。

**観察される作品**: FF7（爆破ミッション開始）, 『スター・ウォーズ』各作, 『マッドマックス 怒りのデス・ロード』, ホメロス『イリアス』, 多くのアクション映画

**例（構造説明のみ。literal として真似しないこと）**:
- [緊張状況] の真っ最中で意識が立ち上がる → 主人公は何かしている → 何のためかは保留
- [追われている] / [追っている] → 理由は後 → 体感速度が冒頭から速い
- [既に起きてしまった事] の直後 → 何があったかの再構成が物語になる

**使い時**: 物語のテンポが命になる時。Q9 に「停滞を避けたい」がある時（桜セッションのような場合）。

**注意**: プレイヤーが状況を理解できないとストレスになる。3〜5 文目で位置情報を出す。

---

### O10. 異常先見せ → 日常

**形**: 冒頭で異常な状況を一瞬見せ、すぐ平常な日常に戻る。読み手は冒頭の異常を**忘れさせられない**。日常が進むほど不穏が増す。

**観察される作品**: Steins;Gate（牧瀬紅莉栖の血だまり→秋葉原の日常）, ひぐらしのなく頃に, ペルソナ5（潜入→逮捕→回想）, 『メメント』, 多くのサスペンス

**例（構造説明のみ。literal として真似しないこと）**:
- [破滅的場面] の数文 → 突然 [ありふれた朝] へ → 違和感が読み手に残る
- [未来の主人公の状態] の予告 → 現在の自分はそれを知らない → 落差が物語を駆動
- [名前/場所/状況] の不一致 → 後でつながる → 種を撒く

**使い時**: 物語に「事件が起きる」確信がある時。プレイヤーに長期の引きを残したい時。

**注意**: 異常先見せの内容を後で必ず回収する。撒いて忘れると不誠実。

---

### O11. 遠未来から振り返る

**形**: 年長になった主人公（または別の語り手）が過去を振り返る形で始める。読み手は冒頭で「これは終わった話」と知る。**結末を知っている安心**と**そこに至る過程の興味**が同居する。

**観察される作品**: 『スタンド・バイ・ミー』, フィッツジェラルド『グレート・ギャツビー』, 桜井亜美『リアル鬼ごっこ』の枠, 一部のアニメ最終回からの回想構造

**例（構造説明のみ。literal として真似しないこと）**:
- [年月が経った] の宣言 → あの時のことを思い出す → 過去に降りる
- [失われた人/場所] の現在 → どう失ったかが物語 → ノスタルジア
- [今となっては] のフレーズ → 当時は知らなかった → 視点の二重性

**使い時**: 物語のトーンに郷愁・喪失・回想を入れたい時。Q5 に「過去」が強くある時。

**注意**: LILIA は進行中の関係を扱うので、**完結した過去** にしすぎない。「ある時期」程度の枠取りに留める。

---

### O12. 神話圧縮

**形**: 物語本編の前に、**世界の昔話** や **遠い過去の出来事** を短く語る。世界観の前提（神、戦争、災厄、起源）を 1 段落で圧縮し、本編の現在に降りる。

**観察される作品**: Undertale（人間と魔物の戦争）, トールキン『指輪物語』プロローグ, FFTactics の歴史, 多くの異世界ファンタジー

**例（構造説明のみ。literal として真似しないこと）**:
- [遠い昔] / [ある時代] の事件 → 結果として今がある → 現在の場面へ
- [伝承] / [噂] の体裁 → 真偽は保留 → 後で本当だと分かる
- [失われた何か] の話 → 今は誰も覚えていない → 主人公だけが触れる

**使い時**: 世界観そのものが物語の柱になる時。Q&A で世界の特異性が確定している時。

**注意**: LILIA の通常セッションでは過剰になりやすい。日常恋愛系では使わない。能力・異世界系の時のみ。

---

## D群: 出会いの作り方（関係の起点をどう置くか）

### O13. 異邦人到来

**形**: 外から人がやってくる。主人公は**迎える側**、または到来そのものを観察する側。日常の場に異物が入った瞬間を冒頭にする。

**観察される作品**: ガルシア=マルケス『百年の孤独』ジプシーの到来, AIR の旅人, 多くの西部劇, 黒澤『七人の侍』, 異世界転生の到来側

**例（構造説明のみ。literal として真似しないこと）**:
- [日常の場] に [見慣れない人物] が現れる → 主人公が反応する → 関係の起点
- [噂] が先に到着 → 本人は遅れて来る → 期待と実物のズレ
- [天候/異変] と同時に来る → 自然との重なり → 運命感

**使い時**: ヒロインが「外から来る」設定の時。Q6 に偶然の到来要素がある時。

**注意**: 主人公が受け身になりすぎる。到来後 3 文以内に主人公の選択を置く。

---

### O14. 救助・発見

**形**: 主人公が倒れている人・困っている人・異常な状態にある人を発見する。主人公の **最初の行動が救助か放置か** で関係が始まる。プレイヤーの初手の選択が関係の起点を作る。

**観察される作品**: 雪山遭難ものの救出, 多くのなろう系の到来, 『君の膵臓をたべたい』のメモ拾い, AIR の浜辺の少女, 桜セッションの想定パターン

**例（構造説明のみ。literal として真似しないこと）**:
- [異常な場所/時間] に [倒れている人物] → 主人公が判断する → 助ける/通過/観察
- [血/傷/異変] の手がかり → ただ事ではない → 関わるリスクを認識
- [助けた直後] ではなく [見つけた瞬間] から始める → プレイヤーの選択を温存

**使い時**: ヒロインが弱った状態で出会う時。Q6 に「助ける/助けられる」要素がある時。

**注意**: いきなり「助けた後」から始めない。**発見の瞬間**から始め、助けるかどうかをプレイヤーに選ばせる。これを飛ばすとプレイヤーが受け身になる。

---

### O15. 異変前夜

**形**: 穏やかな日常がまず確立され、そこに小さな違和が混じる。違和は冒頭では決定的ではない。プレイヤーは「いつもと少し違う」感覚を共有する。やがて違和が大きな出来事に育つ。

**観察される作品**: Fate/stay night の登校風景, 月姫(VN) の朝, CLANNAD の坂道, 多くのギャルゲー, スティーブン・キング『キャリー』導入, ホラー全般

**例（構造説明のみ。literal として真似しないこと）**:
- [いつもの場所] / [いつもの時間] → 1点だけ違う → 小さな引っかかり
- [日常のルーチン] の最中 → 視界の隅の異物 → 言及するか保留するか主人公が選ぶ
- [既知の人物] の様子が少し違う → 理由は語られない → 後で意味が回収される

**使い時**: 物語の核が「日常の崩れ」にある時。多くのギャルゲー・ノベルゲー的展開に向く。

**注意**: 違和を3つ以上重ねると不穏になりすぎる。1〜2点で止める。

---

## 群を超えた組み合わせの指針

冒頭は A群 から 1 つ + D群 から 1 つの組み合わせが安定する。例:

- O1（場所で開く）+ O13（異邦人到来）= 風景 → 訪問者
- O3（音で開く）+ O14（救助・発見）= 不穏な音 → 倒れている人発見
- O4（起床）+ O15（異変前夜）= 起きた瞬間に違和

B群（語り手）と C群（時間）から各 1 つ以下を加えると入れすぎになる。基本は A群×D群の2軸で組む。

C群（時間）は単独で使うことが多い。O9 in medias res を選んだら A群を圧縮する。

## 冒頭で**やらないこと**

- ヒロインの内面を断定する（語り手が主人公でない限り）
- 世界の設定を 5 文以上連続で説明する
- プレイヤーに選択肢を渡さずに 15 文を超える
- 主人公の名前を冒頭で出さない（呼ばれ方は出していい）
- 関係の到達点（恋愛成立、信頼確立）を冒頭で確定する
- 8 原則1「脅かす前に愛させろ」に反して、冒頭で核心の謎を全部見せる

## 冒頭の最後に必ず置くもの

冒頭の最後の 1〜2 文は **プレイヤーの次の動きの余地** を必ず置く。

- 主人公が選択を迫られる物的状況（O14）
- ヒロインの言葉/態度の余韻（O3, O15）
- 場の沈黙、未回答の問い（O5, O15）
- 「どうする？」と書かないが、書かなくても読み手が動ける状態

「→ どうする？」を直接書かない。状況の手掛かりだけを最後に置いて、プレイヤーに動きを返す。
