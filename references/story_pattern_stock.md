# Story Pattern Stock

`references/story_media_stock.md`（50作品カタログ）から抽出した**構造パターン**。
作品単位ではなく、作品をまたいで観察される「形」を集めたもの。

## このファイルと story_media_stock との違い

| ファイル | 役割 | たとえると |
|---------|------|----------|
| `story_media_stock.md` | 作品カタログ（食材図鑑） | 1皿の質を上げる |
| `story_structure_stock.md` | 普遍的構造論（料理の型） | コース全体の流れ |
| `story_pattern_stock.md` | 50作品から見える形（料理人の経験則） | 「この組み合わせは効く」 |

`structure_stock` は理論。`pattern_stock` は経験則。両方ある方が GM の判断は安定する。

## 利用ルール

- 1セッションに1パターンまで。混ぜると形が崩れる。
- パターン名や作品名を本文に出さない（GM の内部ノート）。
- パターンは「次に何をするか」ではなく「今どの形にいるか」を見るために使う。
- 当てはまらないパターンを無理に当てない。
- 例は構造説明のみ。literal として真似しないこと。

## パターン一覧

### P1. 距離の段階的縮小

**形**: 物理的・心理的・情報的距離が、戻れる範囲で少しずつ縮まる。1段縮むたびに**戻れない場所**に近づく。

**観察される作品**: Before Sunrise（一晩の会話）, Normal People（年単位の接近と離反）, 花様年華（互いの配偶者の不在を共有する関係）

**例（構造説明のみ。literal として真似しないこと）**:
- [ヒロインA]: 共有スペースで挨拶する -> 連絡先を持つ -> 仕事帰りに少し待つ -> 弱さを1つ見せる。
- [ヒロインB]: 画面越しの会話 -> 同じ資料を見る -> 同じ沈黙を共有する -> 会わない理由を言う。
- [ヒロインC]: 偶然の同席 -> 同じ移動方向 -> 自分の席を空ける -> 帰る場所の話をする。

**使い時**: 関係が「進んでいない」と感じた時。次の1段だけ進める。

**注意**: 一気に2段以上進めない。戻れなくなる。

---

### P2. 越えてはいけない一線

**形**: キャラが**自分で線を引く**。その線を越えると関係の質が変わる。線は明示的（言葉）でも暗黙的（態度）でもいい。物語は線の周辺で揺れる。

**観察される作品**: 花様年華（不倫にしないという暗黙の線）, Lost in Translation（恋愛にしない線）, ノルウェイの森（線を越えた後の喪失）

**例（構造説明のみ。literal として真似しないこと）**:
- [ヒロインA]の線: 私生活の部屋には入れない。線が揺らぐ: 忘れ物だけ預ける。越える瞬間: 自分から鍵を渡す。
- [ヒロインB]の線: 過去の名前を呼ばせない。線が揺らぐ: 書類上の名前だけ見せる。越える瞬間: 本人が呼び方を選ばせる。
- [ヒロインC]の線: 触れられたくない。線が揺らぐ: 袖を掴む。越える瞬間: 止まれる条件を言って近づく。

**使い時**: 関係の核を作りたい時。線を1本明示すれば、物語はその周りで動き続ける。

**注意**: 線を越えさせる時は意図を持つ。流れで越えると安く見える。

---

### P3. 過去の影が現在を歪める（Ghost in Present）

**形**: 現在のキャラの言動の「不自然さ」が、過去の傷の表れになっている。読者・ユーザーは最初に違和感を感じ、後から理由が見える。

**観察される作品**: Monster（ヨハンの行動の歪み）, Better Call Saul（ジミーの自己破壊）, 3月のライオン（零の対人距離）

**例（構造説明のみ。literal として真似しないこと）**:
- [ヒロインA]: 祝い事の直前だけ返事が遅い -> 後で、祝えなかった約束の記憶が見える。
- [ヒロインB]: 署名欄で必ず手が止まる -> 後で、名前を変えた理由の一部が見える。
- [ヒロインC]: 同じ音を聞くと話題を切る -> 後で、その音に結びつく保留された場面が見える。

**使い時**: キャラの動機が弱いと感じた時。違和感を1つ仕込めば、物語が「謎」として動き出す。

**注意**: 違和感だけ撒いて回収しないと不誠実。Reveal Ladder と連動させる。

---

### P4. 役割からの解放

**形**: キャラが**ある役割**に縛られている（世話役、優等生、記録係、管理者、被害者、加害者）。物語はその役割からの解放、または役割と本人の統合に向かう。

**観察される作品**: Breaking Bad（教師から別の自己へ）, NANA（バンドの顔、恋人、女）, 鋼の錬金術師（賢者の石を求める者）

**例（構造説明のみ。literal として真似しないこと）**:
- [ヒロインA]: 皆の相談役でいる -> [ユーザー]の前では相談できない人になる -> 相談される側から降りる。
- [ヒロインB]: 正確な記録係でいる -> 自分の記憶だけ曖昧になる -> 記録より関係を信じる瞬間が来る。
- [ヒロインC]: 強い側でいる -> 弱さを見せる -> 強さを捨てずに頼る方法を覚える。

**使い時**: キャラが「設定だけで動いている」と感じた時。役割を意識すると、役割を脱ぐ瞬間が物語の山になる。

**注意**: 役割を完全に脱がせると、それまでのキャラが消える。**脱ぎかけ**で止めるのが効く。

---

### P5. 共依存と役割の固定

**形**: 二人の関係が**互いの欠落を埋める形**で固定される。一方が「強い側」、もう一方が「守られる側」になる。物語はその固定が崩れる過程。

**観察される作品**: NANA（ナナとナナの相互依存）, 嵐が丘（ヒースクリフとキャサリン）, ノルウェイの森（直子と緑の対比）

**例（構造説明のみ。literal として真似しないこと）**:
- [ヒロインA]が守られる側で固定される -> [ヒロインA]が[ユーザー]の弱さを見つけ、支える。
- [ヒロインB]が管理する側で固定される -> 管理できない出来事で、頼ることを覚える。
- [ヒロインC]が明るい側で固定される -> 沈黙を許された時、関係が作り直される。

**使い時**: 関係が「片方が強くて片方が弱い」に固定されていると感じた時。役割逆転の瞬間を作る。

**注意**: 固定は安心感を生む。崩しすぎると関係が失われる。崩しと再建をセットで。

---

### P6. 並走から伴走へ

**形**: 二人が同じ方向を見ている（並走）-> 一方が止まる/転ぶ -> もう一方が振り返って戻る -> 一緒に進む（伴走）。横並びの関係から、相互ケアの関係への移行。

**観察される作品**: 3月のライオン（零と川本姉妹）, Frances Ha（女2人の友情の変質）, ハチクロ（ななみー竹本）

**例（構造説明のみ。literal として真似しないこと）**:
- [ヒロインA]と[ユーザー]が別々の仕事を尊重する -> 片方の予定が崩れる -> もう片方が歩幅を変える。
- [ヒロインB]が研究や創作で先を行く -> 途中で止まる -> [ユーザー]が答えでなく時間を渡す。
- [ヒロインC]が日常の儀式を続ける -> 1日だけ途切れる -> 二人で儀式の意味を作り直す。

**使い時**: 関係が「対等で良好だがそれだけ」に見える時。1人を少し止めて、もう1人が戻る瞬間を作る。

**注意**: 振り返らせるためにキャラを過度に弱らせない。**1度の躓き**で十分。

---

### P7. 拒絶の儀式

**形**: キャラが繰り返し相手を拒絶する。**拒絶の仕方**自体に愛着が見えてくる。「来るな」が「行きたくない」に裏返る瞬間。

**観察される作品**: Pride and Prejudice（エリザベスとダーシー）, 嵐が丘（キャサリンの拒絶）, ツンデレ系全般

**例（構造説明のみ。literal として真似しないこと）**:
- [ヒロインA]: 「付き合わなくていい」と言いながら、待ち合わせ場所だけは残す。
- [ヒロインB]: 質問を拒むが、拒み方が毎回少し柔らかくなる。
- [ヒロインC]: 触れられることを避けるが、離れる理由を説明し始める。

**使い時**: 壁の厚いキャラの関係進行に。拒絶は関係の燃料になる。

**注意**: 拒絶を続けすぎるとキャラが嫌われる。3-4回が限界。

---

### P8. 選びと選ばれの非対称

**形**: A は B を選ぶ。B は A を選ぶ。しかし**選び方**が違う。一方は意識的、もう一方は流される形。後でその非対称が露呈する。

**観察される作品**: Before Sunrise / Sunset（連絡しなかった9年）, ノルウェイの森（直子と緑、それぞれ違う選び方）, ロスト・イン・トランスレーション

**例（構造説明のみ。literal として真似しないこと）**:
- [ヒロインA]は「断られなかったから」残る。[ユーザー]は「残ってほしいから」選ぶ。
- [ヒロインB]は仕事の都合で会う。[ユーザー]は会う理由を自分で作る。
- [ヒロインC]は偶然だと思っている。[ユーザー]は何度も選び直している。

**使い時**: 関係が「自然にそうなった」だけで進んでいる時。選びの意識を入れると関係が固まる。

**注意**: 非対称を作るのは難しい。両方が能動的に選ぶ関係も成立する。現在のキャラ次第で選ぶ。

---

### P9. 閉鎖系への外部介入

**形**: 安定した二人の世界（閉鎖系）に、**第三者または出来事**が介入する。介入によって関係の隠れた構造が露呈する。

**観察される作品**: パラサイト（家族と侵入者）, 嵐が丘（ヒースクリフの再来）, NANA（蓮、章司などの介入）

**例（構造説明のみ。literal として真似しないこと）**:
- [ヒロインA]と[ユーザー]だけの連絡習慣に、第三者からの短い依頼が入る。
- [ヒロインB]の仕事場で、二人だけが知る約束を他人が誤解する。
- [ヒロインC]が守っていた小さな生活圏に、古い知人の言葉が戻る。

**使い時**: 二人の世界が安定して止まっている時。介入で動き出す。

**注意**: 介入を頻繁にすると閉鎖系が成立しなくなる。1セッションに1回まで。

---

### P10. 真相の段階開示

**形**: 真相は1度に明かされない。**3-5段階**で、層を剥がすように見えていく。各段階で関係の質が変わる。

**観察される作品**: Monster（ヨハンの過去）, ひぐらしのなく頃に（ループごとの開示）, Disco Elysium（自分の過去の段階的回復）

**例（構造説明のみ。literal として真似しないこと）**:
- [ヒロインA]: 表の職能 -> その職能を選んだ理由 -> 避けている相手 -> 守りたいもの。
- [ヒロインB]: 変わった癖 -> 癖が出る条件 -> 昔の失敗 -> 今の関係で更新される点。
- [ヒロインC]: 共有された事実 -> まだ本人も言語化できない恐れ -> それを隠す防壁 -> 自分で選ぶ言葉。

**使い時**: 物語の長期進行管理に。Reveal Ladder と直結。

**注意**: 段階を一気に2つ進めない。1段ごとに2-3 scene 使って咀嚼する。

---

### P11. 儀式の継続と崩壊

**形**: キャラが何かを**繰り返している**（毎週会う、毎朝同じ挨拶、毎晩同じ片付けをする）。物語はその儀式が**いつ崩れるか**で進む。崩れた瞬間が転機になる。

**観察される作品**: Groundhog Day（同じ日のループ）, ヨコハマ買い出し紀行（喫茶店の日常）, Russian Doll

**例（構造説明のみ。literal として真似しないこと）**:
- [ヒロインA]: 毎回同じ席を選ぶ -> ある日だけ別の席にいる。
- [ヒロインB]: 必ず短い報告を送る -> ある日だけ既読のまま止まる。
- [ヒロインC]: 会う前に同じ買い物をする -> ある日だけ袋の中身が違う。

**使い時**: 日常が安定して進んでいる時、変化を入れたい時。儀式を仕込んでおいて崩す。

**注意**: 崩壊を予告しない。儀式を確立してから、いきなり崩す。

---

### P12. 言葉にならない理解

**形**: 二人が**言葉なしで**理解し合う瞬間がある。説明しない、補足しない、確認しない。読み手はその沈黙を「分かり合った」と読む。

**観察される作品**: 花様年華（廊下ですれ違うだけのシーン）, In the Mood for Love, Lost in Translation（最後の囁き）

**例（構造説明のみ。literal として真似しないこと）**:
- [ヒロインA]が言いかけて止め、[ユーザー]が話題を変えずに同じ沈黙へ残る。
- [ヒロインB]が物を片付ける手を止め、[ユーザー]が答えを急がない。
- [ヒロインC]が帰る前に振り返り、二人とも理由を説明しない。

**使い時**: 説明過多になっている時。1場面、台詞を全部削る。

**注意**: 沈黙には**身体描写**が必要。何もないと空白になる。Wave 1 の「物に感情を預ける」と直結。

---

## パターンの組み合わせ

複数パターンが同時に動くと厚みが出る。

**例（構造説明のみ。literal として真似しないこと）**:
- 厚い防壁の関係 = **P3（Ghost）** + **P2（一線）** + **P10（段階開示）**
- 静かな日常場面 = **P11（儀式）** + **P12（沈黙）**
- 大きな転機 = **P9（介入）** + **P4（役割解放）**

組み合わせは2-3個まで。4個以上は混線する。

## パターンの選び方

セッションを診断する手順:

1. 今、関係はどう動いている？ -> P1, P5, P6 で見る。
2. キャラの動機は弱くないか？ -> P3, P4 で見る。
3. 真相の進行管理ができているか？ -> P10。
4. 場面が薄くないか？ -> P11, P12。

## 出典

このパターンは `references/story_media_stock.md` の50作品から抽出した経験則。
新しい作品をカタログに追加する時、その作品が**新しいパターンを持っているか**を確認すれば、このストックも育つ。
