# プレイヤー入力の書き方

この文書は、プレイ中の入力で **行動・発言** と **内心** を分けるためのルールです。

## 行動・発言

そのまま書きます。
ヒロインが観察したり、聞いたり、反応したりできる情報です。

例:

```text
「夕方なら対応可」と返信する
```

## 内心・思考

行頭、または半角空白/タブの直後に `（...）` または `(...)` で書きます。
これは主人公の内心であり、ヒロインには伝わりません。

例:

```text
（眠いな）返事をする
返事をする （眠いな）
```

混合例:

```text
（めんどくさいけど）「夕方なら対応可」と返信する
```

この場合、ヒロインに伝わるのは「夕方なら対応可」と返信したことだけです。
「めんどくさい」「眠い」などの内心は、ヒロインの台詞や反応には出ません。

## 補足・固有名詞

語の直後に空白なしで `（...）` または `(...)` を書くと、補足として扱われます。
これは行動・発言の一部なので、ヒロインに伝わります。

例:

```text
名刺（裏に連絡先メモ済み）を渡す
[ヒロイン名]さんからのメール（昨日の続き）を確認する
```

## 混合例

```text
（ためらう）名刺（白）を渡す
```

- 内心: ためらう
- 行動・発言: 名刺（白）を渡す

ヒロインは白い名刺を見ます。
ただし、主人公が内心でためらったことは知りません。

## ルールの要点

- 行頭の括弧、または半角空白/タブ直後の括弧は内心
- 語の直後の括弧は補足
- 内心はGMの参考情報にはなるが、ヒロインの認識にはならない
- ヒロインが内心を読んだような台詞や描写は出さない
