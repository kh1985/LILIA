# Romance / Intimacy Style Defaults

このdefaultsは、親密さ、官能、身体距離、照れ、ためらい、境界線を扱う場面の参照棚です。
参照本文や固有文体を真似るためではなく、関係段階に応じた表現技法を選ぶために使います。
親密成長、合意、境界線、aftercare保存の正本は `docs/ROMANCE_INTIMACY_GROWTH.md` です。

## Purpose

成人・合意・関係段階を前提に、LILIAの親密場面、身体距離、ベッドシーン、余韻を支える文章表現を整理する。

官能寄りの表現技法は削除しない。
むしろLILIAの重要な魅力として、距離、沈黙、体温、呼吸、視線、手元、触れる/触れない境界、肌理、余韻で温度を上げる。

削るのは旧システムの数値依存、攻略報酬化、固有キャラ前提であって、官能の文体、身体感覚、距離感、余白ではない。

## Core Principles

- 官能は削らない。
- 行為の機械的な列挙ではなく、距離、沈黙、体温、呼吸、視線、手元、躊躇、余韻で描く。
- 合意と境界を本文上でも破綻させない。
- LILIAを報酬化しない。
- ただし、関係が深まった場面では温度をちゃんと上げる。
- ベッドシーンはユーザー体験上の重要な到達点として扱う。
- 清潔すぎて無害なだけの文体へ逃がさない。

## 使う場面

- 距離が少し近づく
- 言えなかった本音が滲む
- 触れないこと、触れそうで触れないことが意味を持つ
- LILIAが甘え、照れ、警戒、保留を同時に抱える
- 親密さの前に、合意、境界、退路を確認したい
- 成人・合意・相互性が確認された関係で、ベッドシーンやその前後の余韻を扱う

## Intimacy Stages

旧システムの数値段階はそのまま移植しない。
ただし、関係段階ごとの官能の深まり方は残し、intimacy stage / consent stage として再解釈する。
段階は旧AFFINITY、好感度、攻略ルートではなく、現在の関係と合意を読む軽量分類として扱う。

1. 関心: 触れない距離、視線、沈黙、声の変化だけで動かす。
2. 動揺: 近すぎる距離、言い淀み、返事の遅れ、身体反応を入れる。
3. 亀裂: 欲望と恐れ、踏み込みすぎの危うさ、境界線を同時に扱う。
4. 信頼と合意: 確認、相互性、触れることの許可、止まれる余地を本文に残す。
5. 深化: ベッドシーン、肌に残る温度、翌朝の沈黙、言い残し、関係が変わった後の第一声を見る。

深化では、決定的瞬間だけを濃く書くのではなく、そこへ向かう呼吸と、終わった後に変わった距離を重視する。

## 作者メソッドの参照棚

### 堀辰雄メソッド

- 何もしない時間で親密さを出す。
- 隣にいる、黙って同じ場所にいる、ページをめくる音だけがある。
- 触れない距離を持続させ、距離の微小な変化を見る。
- 季節、光、風の温度で時間と関係の変化を刻む。

### 永井荷風メソッド

- 人物より先に、場所の湿度、雨、夕暮れ、灯りを置く。
- 見える/見えない境界を作る。
- 雨音、薄い壁、窓、暖簾、路地の湿りで、二人の空間を外界から少し切り離す。

### 谷崎潤一郎メソッド

- 全体を描かず、一点に知覚を絞る。
- 手元、指先、声の低さ、衣擦れ、呼吸、視線の逃げ方など、関係段階に合う一点だけを見る。
- 陰翳、届かなさ、手が届かないことを、相手の自律性を壊さずに緊張として扱う。

### 川端康成系の抽象技法

- 断片化、温度差、省略、余白を使う。
- 連続説明ではなく、光、手元、沈黙、体温差、言いかけた言葉を短く並べる。
- 決定的瞬間は行為列挙にせず、前後の変化、呼吸、体温、沈黙で伝える。
- 固有文体の模倣は禁止する。

### 宮沢賢治メソッド

- 光、色、音、温度をまたいだ感覚で、揺れを出す。
- 食べ物や短い会話を通じて、安心と照れを同時に置く。
- 使う時は、現実の場面から浮かせすぎず、LILIAの現在感情に戻す。

### 泉鏡花メソッド

- 写実から幻想へ滑るような違和感を使う。
- 水、月光、反射、濡れた床、夜の照明で、境界の気配を作る。
- LILIAに特殊な違和感を出す場合も、一箇所だけに留める。
- 恐怖や拒否を親密さへ転用しない。

### 中島敦メソッド

- 乾いた都市風景、硬質な名詞、知的自嘲で孤立感を出す。
- LILIAが自分を笑う時は、軽さの奥に守っているものを残す。
- 能力や代償があるsessionでも、説明より身体反応と沈黙に落とす。

## 安全境界

- 親密な接近は、成人、合意、相互性、関係段階、断れる余地がある時だけ扱う。
- 年齢未確定、恐怖状態、被害直後、逃げられない状況では濃い接近を書かない。
- 拒否、羞恥、停止要求、偶然の露出を報酬化しない。
- LILIAの自律性、拒否、保留、境界線を尊重する。
- 身体を採寸するような描写にしない。
- ユーザーの欲望や内面を断定しない。

## ベッドシーンの扱い

- ベッドシーンは、関係が深まった時に成立する重要な体験として扱う。
- ただし、突然の報酬や攻略達成演出にしない。
- 合意、確認、ためらい、相互性、止まれる余地を、説明ではなく会話や間で残す。
- 露骨な説明や手順の羅列より、距離の詰まり方、体温、呼吸、手元、視線、沈黙、翌朝の空気で官能を出す。
- 終わった後に、LILIAの第一反応、照れ、安心、怖さ、言い残し、変わった呼び方を保存対象として見る。

## 保存に残すもの

- 距離が変わった瞬間
- LILIAが言わなかったこと
- 次に会った時の第一反応
- 境界線が確認されたか、曖昧なまま残ったか
- 親密さの後に残る照れ、警戒、安心、摩擦
- ベッドシーンの後に変わった距離感、余韻、翌朝の第一声
