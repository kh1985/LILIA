# Scene Situations

## Purpose

このファイルは、LILIA の親密 scene、関係 scene、衝突 scene を立ち上げる際の「どんな状況に置くか」のテンプレート集である。
描写の仕方は `style/defaults/romance.md` の文豪メソッドが正本。本ファイルは状況設定の正本。

## Core Principles

- シチュエーション名を本文や台詞に出さない。世界の出来事として自然に発生させる。
- 既存の Intimacy Stage / Consent Stage / Boundary State と矛盾させない。
- 一場面に複数のシチュエーションを混ぜない。一つだけ選ぶ。
- 同じシチュエーションを連続で使わない。ローテーションする。

## 使う場面

- 親密 scene の入り口を立ち上げる時。
- 関係 scene で温度を一段動かしたい時。
- resume 1ターン目で、前回の温度を思い出させる時。
- 衝突 scene の入り口で、空間の側から圧をかけたい時。

## シチュエーション一覧

### 荷風シチュエーション

- 状況: 雨が降り、二人きりで雨宿りする。路地裏、軒下、車の中、コンビニのイートインの隅。
- 効果: 雨音が外界を遮断し、即席の密室が生まれる。
- 主に効く感覚: 嗅覚、聴覚、触覚
- 仕掛け: 偶然に始まる。二人とも「仕方ない」体で近づくため、意図を言葉にしないまま心の壁が薄くなる。
- 応用例:
  - 帰り道で突然の雨。コンビニの軒下に二人で駆け込む。
  - 夕方の駅。ホームの屋根の下で、電車を一本見送る。
  - 仕事帰り、傘を忘れた相手と一本の傘で歩く。

### 谷崎シチュエーション

- 状況: 暗い部屋。灯りはスマホの光、小さな間接照明、月明かりだけ。
- 効果: 視覚が制限され、触覚と聴覚の解像度が上がる。
- 主に効く感覚: 触覚、聴覚
- 仕掛け: 見えないことで想像が動く。相手の手元、横顔、服の端など一部だけが光に浮かぶ。
- 応用例:
  - 停電したカフェ。スマホで蝋燭代わりに灯す間に手が触れる。
  - 夜の部屋で、間接照明だけ点けて話している。
  - 終電後の暗い駅前。コンビニの灯りが遠くに一つだけある。

### 川端シチュエーション

- 状況: 相手が見えるが、すぐに触れられない距離。ガラス越し、ホーム越し、改札越し。
- 効果: 到達できない近さへの凝視が、関心になる。
- 主に効く感覚: 視覚、温度差
- 仕掛け: 見ているだけ。でも見ていることを相手も知っている。その緊張を、声ではなく距離で置く。
- 応用例:
  - 電車のホーム越しに、向かいのホームの相手が見える。
  - 喫茶店の窓越し。中で本を読んでいる相手を外から見る。
  - 改札の向こう。すでに通ってしまった相手が、振り返る。

### 堀シチュエーション

- 状況: 静かな時間。二人で何もしていない。隣にいるだけ。
- 効果: 何もしない時間の持続が、些細な動きを重くする。
- 主に効く感覚: 視覚、聴覚、時間感覚
- 仕掛け: 安全な時間を置く。だからこそ、いつかこの時間は終わるという予感が残る。
- 応用例:
  - 図書館の窓際の席。並んで本を読んでいる。
  - 公園のベンチ。夕暮れ。会話はなく、風だけがある。
  - 怪我の手当て。相手の肌に触れているが、それは治療だ。でも指先は意識している。

### 鏡花シチュエーション

- 状況: 水場、月光、夜の照明、鏡、ガラス、反射。現実と異界の境界。
- 効果: 反射や輪郭の中に不穏が混じり、安心と違和感が同じ場所に立つ。
- 主に効く感覚: 触覚、視覚、聴覚
- 仕掛け: 先に不快や違和感を一つ置く。その中に LILIA が立つ。落差で、見てはいけないものを見た感覚が残る。
- 応用例:
  - 雨上がりの夜道。水たまりに月が映っている。
  - 夜のプール越し。水面の揺れだけが光る。
  - 銭湯帰り。湿った髪のまま、駅前で別れの会話をする。

### 中島シチュエーション

- 状況: 都市の孤独。深夜のオフィス、無人の駅、閉店後の街。二人だけが残っている。
- 効果: 世界に自分たちしかいない感覚が生まれ、孤独が親密さに転化する。
- 主に効く感覚: 視覚、体感、聴覚
- 仕掛け: 言葉より景色が語る。会話が少ないほど、沈黙と遠景が関係の圧になる。
- 応用例:
  - 終電後の駅。誰もいないホーム。隣に座っている相手の横顔が青白い。
  - 深夜のオフィス。蛍光灯が点いているのは二人の席だけ。
  - 閉店後のカフェ。店員すら帰った後。鍵を持つ相手と二人で残る。

### 賢治シチュエーション

- 状況: 知覚が通常と異なる時間。明け方、夜更け、徹夜明け、能力使用後。
- 効果: 触覚が音に、視線が温度に、匂いが色に寄り、通常の感覚境界が緩む。
- 主に効く感覚: 全感覚
- 仕掛け: 疲労や時間帯で感覚境界が緩んでいる時に、相手と接近する。能力がある session では、能力の余波として扱ってよい。
- 応用例:
  - 徹夜明けの朝。意識がぼんやりした状態で、隣にいる相手の体温を感じる。
  - 雪の夜。冷たい空気の中で、息が見える距離に立つ。
  - 仕事終わりの極度の疲労。視線がうまく定まらない時に、声だけが鮮明に届く。

### 路地裏シチュエーション

- 状況: 大通りから一本入った路地、ビルの隙間、駐車場の暗がり。
- 効果: 公の場のすぐ隣という感覚が、私的な空間を生む。
- 主に効く感覚: 聴覚、視覚、触覚
- 仕掛け: 公的空間の隣にある私的空間を使う。逃げ場と隠れ場が同居し、近づく理由と離れる余地が同時に残る。
- 応用例:
  - 飲み会の帰り道。大通りから一本入った路地で立ち止まる。
  - ビルとビルの間の通路。風が抜ける。
  - 駐車場の隅。車のライトが届かない場所で短く話す。

### 季節 / 時間境界シチュエーション

- 状況: 季節の変わり目、年末年始、誕生日、卒業、引っ越しの前夜など、日常の連続性が一瞬切れる時。
- 効果: いつもと違う日が、判断の重みを変える。
- 主に効く感覚: 時間感覚、季節の匂い、空の色
- 仕掛け: 普段なら言わないことが言える、しないことができる、特殊な日を置く。ただし関係段階を飛ばさない。
- 応用例:
  - 年越しの夜。終電を逃した二人がコンビニで初日の出を待つ。
  - 桜が散り始める日の夕方。歩道に花弁が積もっている。
  - 引っ越し前夜。空っぽになった部屋で最後の夜を過ごす。

## Intimacy Stage との対応

各シチュエーションは、Intimacy Stage に応じて使い分ける。

- 関心 / 動揺 段階: 堀、荷風、路地裏（偶然の接近、安全な距離、街の隙間）
- 亀裂 段階: 谷崎、川端、鏡花（焦らし、届かなさ、境界の気配）
- 信頼と合意 / 深化 段階: 賢治、谷崎、鏡花、季節 / 時間境界（感覚の溶け合い、特別な時間）

中島シチュエーションは段階を選ばないが、関係が成熟してから使うほど残るものが増える。

ファンタジー設定の session（能力 / 異界が出る場合）では、鏡花、中島、賢治を優先する。
現代設定の session では、荷風、谷崎、堀、路地裏、季節 / 時間境界を優先する。

## 安全境界

- シチュエーション名を本文や台詞に出さない。「荷風シチュエーションを発動します」のような自己言及はしない。
- 一場面に複数のシチュエーションを混ぜない。
- 同じシチュエーションを連続で使わない。ローテーションする。
- シチュエーションの直後に衝突 scene を入れない。余韻を壊さない。
- 親密 scene でこれらを使う場合も、`docs/ROMANCE_INTIMACY_GROWTH.md` の Pre-Scene Gate を必ず通る。
- 年齢未確定、恐怖状態、被害直後、逃げられない状況では使わない。
