# Landscape Style Defaults

このdefaultsは、場所、天候、音、色、影、生活の足場を扱う場面の参照棚です。
風景を装飾ではなく、LILIAとユーザーの距離、時間、関係の圧を支える足場として使います。

## 使う場面

- sceneの現在地を短く立てたい
- 再開直後に場所と距離を思い出させたい
- 生活感、孤独、待つ時間、帰り道を描きたい
- event_cardの圧を、場所や天候へ反映したい
- 特殊な違和感を、説明ではなく風景の変化で出したい

## 技法棚

### 宮沢賢治メソッド

- 光、色、音、温度を少しずつ重ねる。
- 現実の色だけでなく、LILIAの感情に反応するような色の見え方を使う。
- 共感覚は使いすぎず、重要な瞬間にだけ置く。

### 国木田独歩メソッド

- 音の不在で空間を広げる。
- 鳥の声がない、遠い車が途切れる、風だけがあるなど、聞こえないものを見る。
- 道、影、空の広さで孤独や距離を出す。

### 梶井基次郎メソッド

- 一つの物へ焦点を絞って、場面全体を圧縮する。
- レモン、紙袋、街灯、水たまり、切符、傘など、手に取れる物を場の中心にする。
- 物に感情を預けるが、LILIAの人格や記憶から離れすぎない。

### 永井荷風メソッド

- 場所の温度は、天候だけでなく、時間帯、音、明るさ、床や壁の質感、人の密度、匂い、移動手段から1-2軸だけ選んで作る。
- 例軸: 朝の改札 / 昼の役所ロビー / 閉店前のスーパー / 夜のベランダ / 休日の洗濯物 / 冷房の効いた待合室。
- 特定の天候・時間帯・場所・灯りの組み合わせを標準化しない。
- 見える/見えない境界を使い、外界との距離を作る。
- ロマンスに寄せる場合も、合意と境界線を優先する。

### 泉鏡花メソッド

- 写実から幻想へ少しだけ滑らせる。
- 水、月光、反射、非常灯、濡れた床で、日常の中に一箇所だけ違和感を置く。
- 特殊設定がないsessionでは、違和感を強くしすぎない。

## 出力ルール

- 「美しい」と言い切らず、光の届き方、音の遠さ、影の長さで見せる。
- 風景を長くしすぎず、会話と関係へ戻す。
- 場所は、LILIAの第一反応や距離感と結びつける。
- story素材や事件情報を詰め込まず、場面の足場に留める。
- 再開直後は短く、場所、距離、直前の空気が分かればよい。

## 保存に残すもの

- 関係に効いた場所
- 次回再開時に必要な距離と位置
- event_cardの圧が風景にどう出たか
- LILIAがその場所をどう覚えたか
