# Heroine Appearance

ヒロインが場面に現れる時、毎回従う。
日常の出会い、再登場、新場面への入場すべてで起動する。

opening_scene.md（初対面）ほど気合いを入れない。
ただし手は抜かない。**繰り返しによってキャラが立ち上がる。**

## 責務（2つ）

1. **その日のヒロイン状態を見せる** — 服装、疲労、機嫌、生活感
2. **キャラのシンボル・仕草を繰り返す** — 同じ人だと体感させる

## 1. 日常状態のチェック

毎回の登場で、以下から **1〜3要素**を出す（全部出さない）:

### 服装
- 季節と場面に合っているか
- ヒロインの生活と矛盾しないか
- 詳細を全部出さない。**1点だけ印象的に**
- 例: シャツの袖を捲っている、靴紐が片方ほどけかけ、髪を結っているか降ろしているか、上着を着たままか脱いでいるか

### 体調・疲労
- 直近の事件・仕事・関係の余韻
- `current/memory.md` の echo と連動: 直前 scene の余韻が体に残っているか
- 例: 目の下が少し青い、声が少し掠れている、姿勢が普段より硬い、コーヒーを飲むペースが速い/遅い

### 機嫌・気分
- 「機嫌が良い」「不機嫌だ」と直接書かない
- 行動で見せる: 椅子の引き方、カップの置き方の音、視線の落とし方、返事の早さ

## 2. シンボル・仕草の繰り返し（必須）

ヒロインを「同じ人」として体感させるため、毎回以下を1つずつ出す:

### Symbol — `lilia/main/profile.md` の「描写の縛り」から

- 描写の縛りに書かれた物的特徴（片方だけのピアス、祖父の万年筆、特定の指輪など）
- 場面ごとに**少しだけ角度を変えて**繰り返す
- 例（同じピアスを違う角度で）:
  - 1場面目: 外の光を受けて一度光る
  - 5場面目: 暗い室内では見えない
  - 10場面目: 雨に濡れている
  - 15場面目: 普段と違って外している（理由は説明しない）

### Gesture — Layer 4（CHINK）の癖

- 手元の癖（カップの縁をなぞる、指を組む、髪を耳にかける）
- 視線の癖（壁を背にする席、伏せ目がち、特定の方向を見ない）
- 距離の癖（一定以上近づかない、急に距離を詰める）
- profile.md の Layer 4 から1つ選んで毎回違う場面で出す

### Drift Guard との連動

`current/story_spine.md` の Drift Guard が増えていれば、それも徐々に織り込む。
新規追加された手がかりは、初出の場面で印象的に出す。

## 3. 文体ルール

### 構造目安（3〜7文）

opening_scene より短め。場面の主役は会話・関係なので、登場描写で長くしない。

参考順序:
1. 周囲の小さな変化（音、空気、光）
2. 登場の輪郭（影、気配、足音）
3. 視覚情報（描写の縛り = Symbol を含む）
4. 動き、姿勢
5. 仕草（CHINK の癖 = Gesture）

### 視覚以外を優先

- 音（足音、鈴、衣擦れ）
- 匂い（雨、コーヒー、生活の匂い）
- 空気の動き
- 距離感

### 主人公との視点（Wave 7）

ヒロインがプレイヤーを見る・近づく・触れる描写では、
`current/protagonist.md` の身体情報を参照する。

- 身長差での視線の高さ
- 距離感（近づき方、触れ方）
- 服装に対する反応（ヒロインがプレイヤーの格好に何かしら反応する場合）

整合性を取るが、本文で「身長180cmのあなたは」のように直接書かない。
描写の中で自然に滲ませる（「彼女は少し見上げて」など）。
`current/protagonist.md` が無い既存セッションでは、この確認をスキップする。

### 比喩は1つまで

2つ以上重ねない。

## 禁止事項

- 顔の詳細列挙（目・鼻・唇）
- 内面の直接描写（"不安そうに"）→ 行動で見せる
- 比喩の重複
- スペック数字
- **毎回同じ描写の繰り返し**（コピペ感が出る）
- **シンボルを毎回フル説明**（読者は覚えている、軽く触れるだけでよい）
- **状態を全部書く**（服装＋疲労＋機嫌の全部出しはくどい、1〜2要素だけ）

## バリエーション

繰り返しは大事だが、コピペにはしない。
同じシンボル・仕草を、違う角度から見せる:

| 角度 | バリエーション例 |
|------|----------------|
| 環境 | 雨/晴れ/夜/朝 |
| 心理 | 余裕/疲労/怒り/沈黙 |
| 距離 | 遠くから見える/すぐ隣 |
| 順序 | こちらが先/彼女が先 |
| 文脈 | 1人で来た/誰かと一緒/急いで来た |

「同じ人だが、毎回違う」を作る。
