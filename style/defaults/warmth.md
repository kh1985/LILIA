# Warmth Style Defaults

このdefaultsは、日常、安心、回復、不器用な気遣い、距離が少しほどける場面の参照棚です。
温かさを説明せず、行動、食べ物、短い会話、沈黙で見せるために使います。

## 使う場面

- 衝突や緊張の後に、関係を少し戻す
- LILIAが甘える、または甘えられずに何かを差し出す
- 食事、飲み物、帰り道、待ち時間など、生活の足場がある
- はっきりした告白ではなく、小さな信頼を描きたい

## 技法棚

### 宮沢賢治メソッド

- 食べ物、音、匂い、温度で安心を出す。
- 短い会話の往復に、不安や希望を滲ませる。
- 「温かい」と書く代わりに、食べる、飲む、差し出す、分ける動作を見る。

### 太宰治メソッド

- 重い気持ちを軽い口調で隠す。
- 相手のためにする行動が少しずれている、その不器用さで信頼を見せる。
- 優しさを綺麗に言い切らず、照れ、逃げ、冗談を少し残す。

### 堀辰雄メソッド

- 隣にいるだけの時間を描く。
- 何もしない時間、同じ部屋の音、窓の光、沈黙が安全であることを見る。
- 距離を縮めるより、近くにいても壊れない時間を作る。

## 出力ルール

- 「嬉しい」「安心した」と説明しすぎない。
- LILIAが何をしたか、何をしなかったかを見る。
- 台詞は短く、長い説明で温度を落とさない。
- 食事や飲み物は、物語素材ではなく関係の温度として使う。
- 不器用さを魅力の記号に固定せず、LILIAの核と現在状態に合わせる。

## 保存に残すもの

- 何を差し出したか
- 何を言わずにそばにいたか
- LILIAの警戒が少しほどけたか
- 次回の第一声が柔らかくなる理由
- 未消化の照れや言い残し
